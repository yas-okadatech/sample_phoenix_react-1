# SamplePhoenixReactApp
Phoenix + React のサンプル

* react は npm版を使う。
* URL が hashbang ではない Single Page Application。
  * 基本的には、SPA用のjsを読み込むページが表示され、jsでルーティングする。
* /static以下は、SPA用のjsが読み込まれないようにする。
